# BRAS Benchmark Data - Sources and Licenses

## Primary Dataset

**BRAS - Benchmark for Room Acoustical Simulation**
- Authors: Lukas Aspoeck, Michael Vorlaender (RWTH Aachen); Fabian Brinkmann, David Ackermann, Stefan Weinzierl (TU Berlin)
- License: **CC BY-SA 4.0** (Creative Commons Attribution ShareAlike 4.0)
- DOI: [10.14279/depositonce-6726.3](https://doi.org/10.14279/depositonce-6726.3)
- Download: https://depositonce.tu-berlin.de/items/version/64
- Date: April 2020

### Files used from this dataset
- Scene descriptions (geometry in SketchUp format)
- Surface descriptions (absorption/scattering CSV files)
- Room acoustic parameters (measured EDT, T20, C80, D50)

### Documentation (not tracked in git, downloadable)
- Database documentation PDF (3.6 MB):
  https://depositonce.tu-berlin.de/items/457e9d28-8ce0-4f57-a6b2-08b1f57efa53

### Associated publication
- L. Aspoeck, F. Brinkmann, D. Ackermann, S. Weinzierl, M. Vorlaender,
  "A benchmark for room acoustical simulation. Concept and database,"
  Applied Acoustics, 2020.
  https://www.sciencedirect.com/science/article/abs/pii/S0003682X20309725

## Supplementary Dataset

**Input and output data for informed room acoustic simulations of the BRAS scene database**
- Author: Lukas Aspoeck (RWTH Aachen, Institute of Technical Acoustics)
- License: **CC BY-NC-SA 4.0** (Creative Commons Attribution NonCommercial ShareAlike 4.0)
- Download: https://publications.rwth-aachen.de/record/803828
- Date: October 2020

### Files used from this dataset
- AC3D geometry models (`scene9.ac`, `scene11.ac`) - exported from BRAS SketchUp files
- RAVEN simulation input files (`.rpf`) - contain source/receiver positions

### Documentation (not tracked in git, downloadable)
- DocumentationInformedSim.pdf (196 KB):
  https://publications.rwth-aachen.de/record/803828/files/DocumentationInformedSim.pdf

### Associated publication
- L. Aspoeck, "Validation of room acoustic simulation models," PhD thesis,
  RWTH Aachen University, 2020.

## Attribution

When publishing results based on this data, cite both:

1. The BRAS database (DOI: 10.14279/depositonce-6726.3)
2. The informed simulations dataset (https://publications.rwth-aachen.de/record/803828)
