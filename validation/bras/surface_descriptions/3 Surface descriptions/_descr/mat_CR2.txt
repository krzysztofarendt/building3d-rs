Author: 					Lukas Aspöck / las@akustik.rwth-aachen.de
Date:						2019/11/21

Notes:						This file contains information about the boundary conditions of the materials (5):
							"mat_CR2_plaster", "mat_CR2_concrete", "mat_CR2_windows", "mat_CR2_floor", "mat_CR2_ceiling"
														
							All relevant values are given in separate files in the csv-format							

							Be aware, that this data is mostly modeled or estimated for the provided room model. 
							Therefore, the dataset does not describe the actual properties of the materials.
